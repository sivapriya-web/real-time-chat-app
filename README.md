# 💬 Real-Time Chat Application

A modern real-time chat application built using **React.js**, **Node.js**, **Express.js**, **Socket.IO**, and **WebSockets**. The application enables instant messaging between multiple users with a responsive and visually appealing user interface.

## 🚀 Features

* Real-time messaging using WebSockets
* Multiple users can chat simultaneously
* User avatars generated automatically
* Emoji picker integration 😊
* Modern Neon Black UI Theme
* Responsive design for desktop and mobile
* Live message updates without page refresh
* Message timestamps
* Online user display

## 🛠️ Technologies Used

### Frontend

* React.js
* Vite
* CSS3
* Socket.IO Client
* React Avatar
* Emoji Picker React

### Backend

* Node.js
* Express.js
* Socket.IO
* CORS

## ⚙️ Installation

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

## ▶️ Run the Application

### Start Backend Server

```bash
cd server
node server.js
```

Server runs on:

```text
http://localhost:3001
```

### Start Frontend

```bash
cd client
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
